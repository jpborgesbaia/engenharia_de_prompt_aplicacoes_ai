# engenharia_de_prompt_aplicacoes_ai
Atividades da Faculdade





Bem-vindo ao repositório da disciplina Engenharia de Prompt e Aplicações em Inteligência Artificial.
Este espaço reúne projetos, experimentos, exemplos de prompts e aplicações práticas utilizando modelos de Artificial Intelligence e Natural Language Processing.

O objetivo é explorar como estruturar prompts eficientes para extrair melhores resultados de modelos de linguagem e construir aplicações baseadas em IA.
